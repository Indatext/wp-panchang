== Description ==

Easily add today's panchang to your blog, powered by Prokerala's astrology API.

= Features =
* Free
* Easy to setup

== Installation ==

1. Upload `panchang` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the short code [panchang] to embed today's panchang into a web page. You can use the following attributes
   to customize the output
   * coordinates: Change the default coordinates from Ujjain.
   * location: Specify the location name (specify if the default coordinates is changed).
   * timezone: Timezone for the location (Default: Asia/Kolkata).
